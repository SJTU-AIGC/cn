上海交通大学 AIGC Class 教育合作计划

“上海交通大学 AIGC Class 教育合作计划” 由上海交通大学 AIGC Class 发起，面向产业界合作伙伴开放，共同推进生成式人工智能（AIGC）在教育、科研与人才培养中的深入应用。

通过校企合作，我们致力于打造集“教学、项目、竞赛、平台建设”于一体的协同育人体系，加速推动 AIGC 技术落地与人才成长。

合作方式
  • 课程共建：联合设计课程内容、派出企业讲师参与教学、评估学生项目成果
  • 项目孵化：提供真实业务问题，支持学生组队研发
  • 实践基地：提供访问、实习机会，或共建实验室
  • 技术平台支持：开放API、模型、训练资源等
  • 资金支持：资助课程开展、竞赛组织、项目孵化及设立冠名奖学金等
企业合作权益
  • 品牌展示：企业LOGO将展示在官网、宣传材料中
  • 人才优选：优先获取学生实习、就业推荐机会
  • 成果共创：优秀项目可进入企业孵化流程
  • 协议合作：可签署产学合作协议、冠名课程等
